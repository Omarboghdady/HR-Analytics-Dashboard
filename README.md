📊 HR Analytics Dashboard | Power BI Project

🎯 Project Overview
This project presents a comprehensive HR Analytics Dashboard built using Power BI Desktop to transform raw HR data into actionable insights that support strategic workforce decision-making.

The dashboard analyzes key HR dimensions including:

Workforce structure
Employee performance
Salary distribution
Attendance behavior
Recruitment efficiency
Department-level insights

🧰 Tools & Technologies

Power BI Desktop
DAX (Data Analysis Expressions)
Power Query (Data Cleaning & Transformation)
Data Visualization Techniques

🔍 Key Findings

🔴 Critical Issues

High turnover in Finance department (26%)
High number of low performers (98 employees)
IT department shows salary-performance imbalance

🟡 Areas for Improvement

Marketing absence levels require attention
Recruitment conversion rate is very low (1.96%)
Seasonal turnover patterns in August & December

🟢 Positive Insights

Operations department shows strong performance
Workforce distribution is balanced
Overall absence rate remains relatively low (7%)

💡 Final Insight

While the organization maintains a stable workforce structure, there are key challenges in:
Employee retention
Performance optimization
Recruitment efficiency
Addressing these areas can significantly improve productivity, reduce costs, and enhance overall organizational effectiveness.
